# Circuit Simulator

Python-based circuit simulator capable of performing transient analysis of electrical circuits using Modified Nodal Analysis (MNA) and LU decomposition.

## Contributors

- Tanmay Singh
- Vaibhav Poddar

## Features

- Modified Nodal Analysis (MNA)
- LU Decomposition solver
- Voltage and current waveform visualization
- Interactive GUI for circuit simulation

## Technologies Used

- Python
- NumPy
- Tkinter
- Eel
- HTML/CSS
- Chart.js
